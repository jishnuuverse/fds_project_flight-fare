# ✈️ Flight Fare Prediction — A Data Science Case Study

A Fundamentals of Data Science (FDS) college course project exploring flight
ticket pricing data and building regression models to predict fares based
on airline, route, and journey-related features.

---

## 📌 Overview

This project analyzes a flight fare dataset to uncover the factors that
drive ticket prices — such as airline, number of stops, duration, and
departure/arrival times — and builds machine learning models to predict
fares from these features.

---

---

## 🔍 Approach

1. **Data Exploration (EDA)** — Understand distributions, missing values,
   and relationships between features and fare
2. **Data Cleaning & Preprocessing** — Handle missing values, encode
   categorical variables, engineer date/time features
3. **Model Building** — Train regression models to predict flight fares
4. **Evaluation** — Compare models using standard regression metrics
5. **Presentation** — Findings summarized in the accompanying case-study
   slide deck
